## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Authentication**: NextAuth.js
- **Database**: MongoDB with Prisma ORM
- **Styling**: Tailwind CSS
- **UI Components**:
  - NextUI
  - Radix UI
  - Framer Motion
- **Payment Processing**: Razorpay
- **Deployment**: Vercel
- **Analytics**: Vercel Analytics

- Google OAuth integration
